Give an LLM a text quest from Space Rangers: read the situation, pick an action, repeat for 10-50 turns. Run the same quest with five agent architectures - from a bare prompt to a planning agent with tools. Compare success rates, cost, and failure modes.

Everything is open source: framework, data, configs. The quest engine is space-rangers-quest by roginvs.
